الحسابات المربوطة
إدارة حسابات Gmail المتصلة بـ MailVault
🔗 خطوات ربط Gmail
اتبع الخطوات دي عشان تربط حسابك بالموقع
  1. 1
    روح على console.cloud.google.com وعمل Project جديد باسم MailVault
  2. 2
    من القائمة فعّل Gmail API من خلال APIs & Services → Library
  3. 3
    روح OAuth consent screen، اختار External، أدخل اسم التطبيق وإيميلك
  4. 4
    من Credentials اعمل OAuth 2.0 Client ID → اختر Web application
    أضف Redirect URI: https://your-domain.com/callback
  5. 5
    احتفظ بـ CLIENT_ID و CLIENT_SECRET وحطهم في GitHub Secrets
  6. 6
    في GitHub Actions الـ workflow يعمل deploy للباك-إند على Pterodactyl عن طريق PTERODACTYL_API_KEY
✏️ رسالة جديدة ✕
